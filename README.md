# coffee-shop-database-system
# Coffee Shop Database Management System

A relational database project developed for managing customer orders, inventory, product listings, and sales reports in a coffee shop business environment.

## Technologies Used
- Microsoft Access
- SQL
- ER Modeling
- Query Design

## Core Features
- Customer order records
- Inventory tracking
- Product database
- Sales reporting
- Relational table normalization
- Query generation

## Project Purpose
Developed to demonstrate database design, entity relationship modelling, and structured query implementation.
